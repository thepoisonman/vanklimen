¿Por qué hice una página?

  • Para poder invitarte a redireccionar sin decir link en bio.

  • Para no estar compitiendo al ritmo de un scroll.

  • Para tratar a volver al gusto de la vieja internet.

  • Para no perderme en la ansiedad y tratar de que a mis amigxs no les pase lo mismo.

  • Para una atención más sana y no inmediata.

  • Para olvidarme en branding porque me resulta un embole,

  • Para que sigamos siendo múltiples y gregarios.

  • Para tener un diario de artista e invitar a que varios lo tengan.

  • Para sorprenderme.

  • Para defender mi visión y entender que puede ser permeable.

  • Para ser consciente de mi recorrido: de dónde vengo y quiénes me transitaron.

  • Para compartir lo que sé y, sobre todo, de lo que estoy aprendiendo.

  • Para que mis amigxs publiquen sin importarles la mirada del algorritmo.

  • Para intentar poder publicar algo sin estar pendiente de la hora y de su llegada:

  • Para alejarme de la estrategia.

  • Para que construyamos las estretegias.

  • Para que sea un portal.

  • Para que sea una expansión.

  • PAra declararle la guerra a la forma -usando la forma-.

  • Para que sea imperfecta.

  • Para que sea tan inútil que alguien le encuentre una utilidad.

  • Para que se me vaya de las manos.

Bueno, eso…
